\section{Постановка и результаты экспериментов}

Эксперименты проводилимь на 6 ядерном компьютере с процессором Intel Core i7 980x с частотой 3.33GHz, оперативной памятью 12 Гб, ОС - Calculate Linux 10.4 x64. 

Для сборки приложения с библиотекой tbb, необходимо, что бы компилятор поддерживал лямбда выражения из нового стандарта c++0x. На данный момент, этот стандарт поддерживается в следующих продуктах:
\begin{itemize}
	\item Intel С++ Compiler 11.0
	\item Microsoft Visual Studio 2010
	\item GCC 4.5
\end{itemize}
